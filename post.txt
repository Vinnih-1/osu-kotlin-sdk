[size=150][b]Installation[/b][/size]

Gradle (Kotlin DSL)

This project needs to be used with Coroutines, you can see more information [url=https://github.com/Kotlin/kotlinx.coroutines]here[/url].

[code]
implementation("io.github.vinnih-1:osukdk:VERSION")
[/code]

[size=150][b]Quickstart[/b][/size]

[code]
fun main() = runBlocking {
    val api = Authorization(YOUR_CLIENT_ID, YOUR_CLIENT_SECRET).create()
    val user = api.getUser(21009314).also { println(it.username) }
}
[/code]
